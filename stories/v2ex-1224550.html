大家好，我是 ZGI 的产品运营 Eddie 。

最近我们把 ZGI 开源了，想来 V2EX 跟大家分享一下，也欢迎大家体验后直接提意见。

ZGI 不是一个单纯的聊天工具，更像是一个给 AI Agent 用的运行平台。简单说，就是把模型、知识库、工作流、数据库、权限、Token 这些东西放在一起，让 Agent 不只是能聊，还能在企业场景里真正跑起来。

目前 ZGI 已经接入了不少大模型，比如：

GPT 克洛德 深度搜索 奎文 双子座

如果你平时需要在多个模型之间切换，不想来回打开不同平台，可以直接在 ZGI 里配置和使用。

除了模型调用，目前也支持一些基础能力：

文档知识库 / RAG 可视化 Agent 编排 工作流 Skills 数据库查询 API / Webhook Token 统计 成员权限管理

我们最近也在做一个自己觉得挺有意思的功能：用自然语言操作平台。

比如你可以直接跟 AI 说：

帮我创建一个客服 Agent 把这个 Agent 换成 Claude 新建一个知识库，把这个文档加进去 帮我搭一个定时汇总日报的工作流

以前这些都要自己点页面、找配置项，现在可以让 AI 帮你做一部分操作。

项目刚开源，还有很多地方肯定不完善。 如果大家愿意体验一下，帮忙看看哪里不好用、文档哪里看不懂、功能设计哪里不合理，对我们会很有帮助。

GitHub： https://github.com/zgiai/zgi 官网： https://www.zgi.ai/

不敢说我们比谁更好，Dify 、Coze 、n8n 都有各自很强的地方。我们现在更想把“企业里 Agent 怎么稳定跑起来”这件事做好。

欢迎大家拍砖。
