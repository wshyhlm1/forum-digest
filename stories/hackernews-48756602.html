Kimi K2.7 代码已在 GitHub Copilot 中公开

Kimi K2.7 Code is generally available in GitHub Copilot

AI 作者 unliftedq 热度 342 回复 143 2026/07/02 12:32
打开原帖 打开外部链接 JSON

摘要

  • Kimi K2.7 Code 开源模型现已在 GitHub Copilot 中正式可用，是首个作为可选选项提供的开源权重模型。
  • 该模型由 GitHub 托管在 Microsoft Azure 上，按使用量计费，为开发者提供更低成本的编程工作流选择。
  • 目前正逐步向 Copilot Pro、Pro+ 和 Max 计划用户推出，后续将扩展至 Business 和 Enterprise 版本。

看点

  • 首个在 Copilot 模型选择器中提供的开源权重模型
  • 支持 VS Code、Visual Studio、JetBrains、Xcode 等多个主流开发环境
  • 企业版默认关闭，需管理员手动启用并评估安全合规要求
  • 按提供商列表价格进行基于使用量的计费
  • GitHub 将持续监控模型质量和性能表现

帖子介绍了 Kimi K2.7 Code 开源模型在 GitHub Copilot 中的正式可用，涉及 AI 编程助手、开源模型、开发工具集成等核心科技话题。
